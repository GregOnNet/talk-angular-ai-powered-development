# The Problem with Prompts

Missing information leads to poor results

<div class="mt-6">

**Example prompt:**
- Create a component that allows to rate a product from 1-5
- Create a service that allows to update the rating

</div>

<div class="mt-6">

**Missing:**
- How should the rating be displayed?
- When shall the API be called?
- What is the API endpoint?
- What coding standards apply?

</div>

::right::

<div class="mt-6">

**Solution: Rules Files**

- Requirements change and are unique
- Quality criteria stay the same
- Fit all information in prompts
- Apply standards to every prompt
- Share instructions across team

</div>

<div class="mt-8 opacity-70">

This is how AI makes your coding standards a living standard!

</div>
