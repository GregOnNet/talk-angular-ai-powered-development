# What Is the MCP Server?

Model Context Protocol server for Angular

<div class="mt-6">

- **Angular CLI integration** - Install via Angular CLI
- **Tool calls** - Search Angular Documentation
- **Best practices** - Get most up-to-date guidance
- **Official support** - Maintained by Angular team
- **Protocol standard** - Uses Model Context Protocol

</div>

<div class="mt-10 opacity-70">

It allows tool calls to search the Angular Documentation and get most up to date best practices from https://angular.dev

</div>
